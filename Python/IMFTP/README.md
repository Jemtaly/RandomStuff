# IMFTP

A file transfer tool and an instant messager.

## Usage

```
usage: imftp [-h] [--server [IP] | --client IP] [--port PORT] (--send [FILENAME] | --recv [FILENAME] | --talk)
             [--size SIZE]

A file transfer tool and an instant messager.

options:
  -h, --help         show this help message and exit
  --server [IP]      run as a server (default)
  --client IP        run as a client
  --port PORT        set port (2351 by default)
  --send [FILENAME]  send file
  --recv [FILENAME]  receive file
  --talk             instant messager
  --size SIZE        set the initial value
```

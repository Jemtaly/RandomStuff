# BrainFuck

A brainfuck interpreter written in bash.

## Usage

### Print "Hello World!"

```bash
echo '++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.' | bf
```

or

```bash
echo '++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.' >> a.txt
bf a.txt
```

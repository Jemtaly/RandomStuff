# BrainFuck

A brainfuck interpreter written in bash.

## Usage

### Print "Hello World!"

```bash
echo '++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.' >> a.txt
./bf.sh a.txt
```

or

```bash
echo '++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.' | ./bf.sh -
```
